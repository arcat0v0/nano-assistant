# nano-assistant v0.3.0

## Highlights

- Added knowledge-source skills for ArchWiki, Debian Wiki, and Red Hat docs
- Added self-management flows for skill installation, MCP config updates, and `MEMORY.md`
- Added `pty_shell` for interactive CLI automation
- Added Windows path/shell support and basic interactive command support
- Builtin skills now report the binary version and reject direct source edits

## User-facing changes

- `na skills list` now shows builtin skills with synced version information
- Skill installs can be auto-detected and reloaded without restart
- MCP server changes in `config.toml` can be auto-reloaded after edit
- On Windows, `pty_shell` supports prompt/response flows through stdin/stdout pipes

## Windows notes

- Configuration path: `%APPDATA%\nano-assistant\config.toml`
- Memory path: `%APPDATA%\nano-assistant\MEMORY.md`
- Shell execution uses `cmd /C`
- Interactive command support is suitable for prompts, confirmations, and password entry
- Full-screen TUI compatibility is still limited until native ConPTY support is added

## Verification

- `cargo test -q`
- Result: unit tests passed, integration tests passed, 0 failures
